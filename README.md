## Hardware

-   **CPU:** Intel 8th m3-8100y
-   **Graphics:** Intel HD Graphics 615
-   **RAM:** 8G LPDDR3
-   **Memory:** WD Black SN750 250GB M.2 SSD
-   **Network:** Gigabit Ethernet & WIFI 802.11 AC, 2.4G & 5G
-   **Dimension:** 115mm * 78mm * 14 mm
